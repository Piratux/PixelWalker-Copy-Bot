## MIDI info

MIDI is a file format that stores music note data.

Here you can import a MIDI file's tracks to PixelWalker.

## Supported MIDI families

An attempt has been made to map similar sounding instruments to closest PixelWalker representation.

Some MIDI families are explicitly checked. Unknown MIDI families are mapped to piano.

- PW Guitar - MIDI family: [guitar, bass]
- PW Drums - MIDI family: [drums]
- PW Piano - MIDI family: the rest of families

## PixelWalker limitations

PW does not support varying note length and volume for each note.

For best results, try songs that have short notes and all instruments have similar volume throughout the song.
