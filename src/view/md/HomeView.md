## General info

This web app lets you host Copy Bot for PixelWalker.

## Feature overview

Features accessible via commands:

- Copy/paste selected region
- Move selected region
- Mask blocks when pasting (such as only pasting background blocks)
- Repeated paste of selected region
  - Smart repeated paste - switches/portals/etc. get auto incremented based on placed pattern (useful for building switch worlds)
- Undo/redo blocks placed by bot
- Import PixelWalker worlds via link
- Replace all selected blocks
- Perform math operations with selected block parameters (ex.: add 5 to all counters)

Features accessible via bot page:

- Export PixelWalker world to .eelvl file
- Import PixelWalker world from .eelvl file
- Import PixelWalker world from .midi file
- Import PixelWalker world from .png file
- Import PixelWalker world from EER (Everybody Edits Rewritten) world
- Find locations of specific blocks in the world
- Minimap teleporter - click on minimap to teleport there
