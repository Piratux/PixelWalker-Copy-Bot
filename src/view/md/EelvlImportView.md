## Import info

EELVL is a file format that was used by Everybody Edits (EE).

Here you can import EE worlds from `.eelvl` files to PixelWalker.

## Differences

Compared to EELVL, PixelWalker doesn't have:

- Block for picked up gold/blue coin [110, 111]
- Trophy [478 - 480, 484 - 486, 1540 - 1542]
- Gold gate/door [200, 201]
- Golden easter egg decoration [1591]
- Green space decoration [1603]
- NPC [1550 - 1559, 1569 - 1579]

All missing blocks are replaced with signs of block name.

Note: numbers in `[]` brackets represent EELVL block IDs.
